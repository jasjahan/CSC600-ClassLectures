```
npm install tslab
```

```
npm install
```

Instructions from Lawrence:

1. $ pip3 install jupyter
2. $ npm install -g tslab
3. Put the globally-installed "tslab" in your $PATH. Restart your shell.
4. $ tslab install
5. $ jupyter notebook

And to add Jupyter Lab:

6. $ pip3 install jupyterlab
7. $ jupyter lab
